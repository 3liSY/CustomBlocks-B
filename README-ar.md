# 🧱 CustomBlocks
 
> حوّل أي صورة إلى بلوك ماينكرافت — من رابط URL أو GIF وغيرها. بدون أي كود.
 
**ماينكرافت:** 1.21.1 &nbsp;|&nbsp; **اللودر:** Fabric &nbsp;|&nbsp; **المطوّر:** [Srb Gamer](https://www.youtube.com/@SrbGamerr)
 
---
 
## ✨ المميزات
 
- 🖼️ **صورة على بلوك** — ضع أي صورة على بلوك عبر رابط URL (يدعم GIF)
- 🎨 **تكستشر لكل وجه** — ضع تكستشر مختلفة على كل وجه من وجوه البلوك
- 🟨 **أيتمات ملونة** — أيتمات مربع ومثلث بألوان أسود وأصفر وأخضر مع إعادة تلوين ديناميكية
- 💡 **التحكم بالإضاءة** — اضبط مستوى الإضاءة لكل بلوك
- 🔊 **الصوت والصلابة** — خصص صوت كسر البلوك ومستوى صلابته
- 📐 **أشكال مخصصة** — عيّن أو أضف أو احذف أو احفظ أشكال الـ collision والمظهر
- ↩️ **تراجع / إعادة** — دعم كامل للتراجع والإعادة على جميع التغييرات
- 📦 **1028 سلوت بلوك** — سعة ضخمة جداً بدون أي مشكلة في الـ registry عند إعادة التشغيل
- 🖥️ **واجهة داخل اللعبة** — محرر بصري ولوحة أدمن، بدون أوامر
- 📤 **تصدير واستيراد** — صدّر بيانات البلوكات واستورد فولدرات كاملة
- 🔁 **نسخ البلوكات** — استنسخ أي بلوك مع جميع إعداداته

## ⚙️ الأوامر
 
جميع الأوامر تبدأ بـ `/customblock` أو `/cb`.
 
| الأمر | الوصف |
|---|---|
| `create <id>` | أنشئ بلوكاً جديداً من رابط URL |
| `delete <id>` | احذف بلوكاً |
| `rename <id> <newname>` | أعد تسمية بلوك |
| `retexture <id> <url>` | غيّر تكستشر بلوك موجود |
| `give <player> <id>` | أعطِ بلوكاً لأحد اللاعبين |
| `resize <id> <scale>` | غيّر حجم تكستشر البلوك |
| `setglow <id> <0-15>` | اضبط مستوى الإضاءة |
| `sethardness <id> <value>` | اضبط صلابة البلوك |
| `setsound <id> <sound>` | اضبط صوت البلوك |
| `setshape <id> <shape>` | عيّن شكل البلوك |
| `addshape / removeshape / clearshape` | إدارة مكونات الشكل |
| `saveshape / loadshape` | احفظ وأعد استخدام الأشكال |
| `setcollision on/off` | فعّل أو عطّل الـ collision |
| `undo / redo` | تراجع أو أعد آخر تغيير |
| `dupe / duplicate <id>` | استنسخ بلوكاً مع بياناته |
| `export <id>` | صدّر بيانات بلوك إلى ملف |
| `importfolder <path>` | استورد بلوكات من فولدر |
| `list` | اعرض جميع البلوكات المسجّلة |
| `gui / editor` | افتح محرر البلوكات داخل اللعبة |
| `admingui` | افتح لوحة الأدمن |
| `settabicon <id>` | عيّن أيقونة تاب الكريتف |
| `reid <id> <newid>` | غيّر الـ ID الخاص ببلوك |
| `reload` | أعد تحميل جميع بيانات البلوكات |
| `help` | اعرض جميع الأوامر المتاحة |
