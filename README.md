# 🧱 CustomBlocks

> Turn any image onto Minecraft blocks — from a URL, a GIF, etc. No coding needed.

**Minecraft:** 1.21.1 &nbsp;|&nbsp; **Loader:** Fabric &nbsp;|&nbsp; **Author:** [Srb Gamer](https://www.youtube.com/@SrbGamerr)

---

## ✨ Features

- 🖼️ **Image-to-block** — Apply any image to a block using a URL (supports GIFs)
- 🎨 **Per-face texturing** — Apply different textures to each face of a block
- 🟨 **Colored items** — Black, Yellow, and Green square & triangle items with dynamic recoloring
- 💡 **Glow control** — Set custom light levels per block
- 🔊 **Sound & hardness** — Customize block sounds and break hardness
- 📐 **Custom shapes** — Set, add, remove, or save collision/visual shapes
- ↩️ **Undo / Redo** — Full undo and redo support for all block changes
- 📦 **1028 slot blocks** — Really massive capacity with no registry mismatch on reload
- 🖥️ **In-game GUI** — Visual editor and admin panel, no commands required
- 📤 **Export & Import** — Export block data and import entire folders
- 🔁 **Duplicate blocks** — Clone any block with all its settings intact

## ⚙️ Commands

All commands start with `/customblock` (or `/cb`).

| Command | Description |
|---|---|
| `create <id>` | Create a new custom block from a URL |
| `delete <id>` | Delete a block |
| `rename <id> <newname>` | Rename a block |
| `retexture <id> <url>` | Change the texture of an existing block |
| `give <player> <id>` | Give a block to a player |
| `resize <id> <scale>` | Resize the block's texture |
| `setglow <id> <0-15>` | Set light emission level |
| `sethardness <id> <value>` | Set block hardness |
| `setsound <id> <sound>` | Set block break/place sound |
| `setshape <id> <shape>` | Set the block's visual/collision shape |
| `addshape / removeshape / clearshape` | Manage shape components |
| `saveshape / loadshape` | Save and reuse shapes |
| `setcollision on/off` | Toggle collision for a block |
| `undo / redo` | Undo or redo the last block change |
| `dupe / duplicate <id>` | Duplicate a block with all its data |
| `export <id>` | Export block data to file |
| `importfolder <path>` | Import blocks from a folder |
| `list` | List all registered custom blocks |
| `gui / editor` | Open the in-game block editor |
| `admingui` | Open the admin GUI panel |
| `settabicon <id>` | Set the creative tab icon |
| `reid <id> <newid>` | Re-ID a block |
| `reload` | Reload all block data |
| `help` | Show all available commands |
